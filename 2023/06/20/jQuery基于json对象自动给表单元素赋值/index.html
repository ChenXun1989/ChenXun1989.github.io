jQuery基于json对象自动给表单元素赋值

为了提高前端小伙伴的开发效率，造了个基于json对象根据表单元素的name属性自动赋值的轮子

注意： 表单的name属于与json对象的属性名为一致，保持继承链。
例如 input name=’a.b.c’ 表示json对象里面的a属性里面的b属性的c属性。
